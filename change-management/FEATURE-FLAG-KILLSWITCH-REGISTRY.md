# Feature Flag Kill-Switch Registry (Normatif)

Objectif: lister explicitement les rares feature flags autorisés à rester permanents.

Règles:

* toute entrée DOIT inclure un owner, une justification, et un runbook de rollback
* toute entrée DOIT être revue au minimum tous les 90 jours
* une entrée sans owner actif DOIT être supprimée
